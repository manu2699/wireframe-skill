# wireframe-preview (deprecated)

> This package has been renamed and republished as **[proto-frames](https://www.npmjs.com/package/proto-frames)**.

## Migrate

Uninstall the old package and install the new one:

```bash
npx wireframe-preview uninstall
npx proto-frames install
```

If you registered the MCP server:

```bash
npx wireframe-preview uninstall mcp
npx proto-frames mcp <platform>
```

## Why

The package was renamed to `proto-frames` to better reflect its scope — mid-fidelity wireframes and clickable prototypes, not just static previews.

All features, the JSON model schema, and MCP tools are identical. No breaking changes.
